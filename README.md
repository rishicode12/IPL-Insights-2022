# IPL 2022 Data Analytics Dashboard

## 1. Project Title
IPL 2022 Match & Player Performance Analysis

This data analytics project explores match outcomes, team performance, player statistics, and match conditions from the Indian Premier League (IPL) 2022 season using Python and data visualization techniques.

## 2. Project Objectives

The main goals of this project are:

Understand team performance patterns across the season

Analyze the impact of toss decisions on match results

Identify top performing batsmen and bowlers

Find factors influencing winning probability

Present insights using clear and interactive dashboards

## 3. Dataset Used

Source: IPL 2022 Match Dataset (CSV format)

Dataset File: IPL_data_2022.csv

Kaggle Link:
IPL 2022 Dataset

Dataset Includes:

Match details (date, venue, teams)

Toss decision and toss winner

Match winner

Player performance

Runs, wickets, overs

Team statistics

## 4. Key Questions (KPIs)

This project answers the following analytical questions:

1.Which team performed best in IPL 2022?

2.Which team won more matches while defending vs chasing?

3.Who were the top bowlers by wickets taken?

4.Who were the top batsmen by runs scored?

5.Which player won the highest number of Man of the Match awards?

6.Which team won the highest number of matches?

7.Who was the highest run scorer in IPL 2022?

8.Does winning the toss increase winning chances?

9.Which team won the highest number of tosses?

## 5. Process Followed
Step 1: Data Cleaning

Removed missing values

Standardized team names

Converted data types

Handled inconsistent records

Step 2: Data Processing

Created derived columns (win type, match margin)

Aggregated team statistics

Calculated win percentages

Prepared player performance metrics

Step 3: Exploratory Data Analysis (EDA)

Performed analysis using:

Pandas

NumPy

Step 4: Visualization

Charts created using:

Matplotlib

Plotly

Seaborn

Visualizations Include:

Team performance comparison

Top players ranking

Toss impact analysis

Venue-based performance insights
